What is the Difference Between Border, Margin and Padding

Margin, border and Padding in coding refer to the space between elements in a design. Margin, refers to the space between the border of a design and the next element of your design.
The padding is the inside space of the design, between the border and the content. The easiest way to understand this is to see it in a picture.

As you can see in the first picture, the margin gives us space between two objects,
the border provides and outline of the object and the padding provides space between the border and the content that is inside the object.
The second image shows the Margin and padding when two objects are involved.

How to Use Margins, borders and Padding in CSS

To add a margin to an element, four margins need to be declared, representing each side of the element.
These would be Margin-top, Margin-right, Margin-bottom and margin left.
The margin property allows for you to set all the margins to one number, however, if you wanted to have each margin have a seperate length or percentage you can set them individually.
To do this you would use the code "margin: 20x 24x 35x 100x;" with x representing which unit of measurement you want to use.
The order goes around your object clockwise. Padding uses the same basic code as margins, you just have to replace the word margin with padding.
Borders also use a basic code and can be stylised easily too.
